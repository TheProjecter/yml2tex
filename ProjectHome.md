yml2tex is a Python script which will generate a LaTeX Beamer presentation out of a YAML file.

### Readme ###
[The README can be viewed here](http://github.com/arthurk/yml2tex/tree/1.2/README.markdown)

### Installation ###

```
easy_install yml2tex
```
